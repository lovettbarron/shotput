  <name>Task 1: Initialize project with dependencies, build config, and license</name>
  <files>package.json, tsconfig.json, tsup.config.ts, vitest.config.ts, LICENSE, src/types.ts</files>
  <action>
1. Create `package.json` with:
   - name: "shotput", version: "0.1.0", license: "MIT", type: "module"
   - bin: { "shotput": "./dist/index.js" }
   - scripts: build (tsup), start (node dist/index.js), dev (tsup --watch), test (vitest run --reporter=verbose), license-check (license-checker --onlyAllow 'MIT;Apache-2.0;ISC;BSD-2-Clause;BSD-3-Clause;0BSD;CC0-1.0;Unlicense')
   - dependencies: @modelcontextprotocol/sdk@^1.27.1, playwright@^1.58.2, zod@^3.25.0
   - devDependencies: @types/node@^22.0.0, tsup@^8.0.0, typescript@^5.0.0, license-checker@^25.0.0, vitest@latest

2. Run `npm install` to install all dependencies

3. Run `npx playwright install chromium` to install the browser binary

4. Create `tsconfig.json`:
   - target: "ES2022", module: "ESNext", moduleResolution: "bundler"
   - strict: true, esModuleInterop: true, skipLibCheck: true
   - outDir: "dist", rootDir: "src"
   - include: ["src/**/*"], exclude: ["node_modules", "dist"]

5. Create `tsup.config.ts` per research:
   - entry: ["src/index.ts"], format: ["esm"], target: "node22"
   - outDir: "dist", clean: true, sourcemap: true
   - external: ["playwright"] (CRITICAL: Playwright has native bindings, must not be bundled)
   - dts: false (not a library)

6. Create `vitest.config.ts`:
   - import { defineConfig } from 'vitest/config'
   - test.globals: true, test.testTimeout: 30000 (Playwright tests need time)

7. Create `LICENSE` file with MIT license text, copyright 2026

8. Create `src/types.ts` with shared interfaces:
   ```typescript
   export type WaitStrategy = "networkidle" | "domcontentloaded" | "load";

   export interface CaptureParams {
     url: string;
     fullPage: boolean;
     format: "png" | "jpeg";
     quality?: number;
     width: number;
     height: number;
     scale: number;
     outputDir?: string;
     filename?: string;
     wait: WaitStrategy | number;
     autoScroll: boolean;
     timeout: number;
   }

   export interface CaptureResult {
     filePath: string;
     buffer: Buffer;
     format: "png" | "jpeg";
     width: number;
     height: number;
     warning?: string;
   }
   ```

9. Create a minimal `src/index.ts` placeholder (just `console.error("shotput starting...")`) so tsup can build.

10. Run `npm run build` to verify tsup builds successfully.

11. Run `npm run license-check` to verify all dependency licenses are MIT-compatible (QUAL-04).
