1. Implement inspectPage in src/inspect.ts:
   - Use getBrowser() to get browser instance, create context with viewport (width default 1280, height default 720)
   - Navigate to URL with specified wait strategy (default "networkidle") and timeout (default 30000)
   - Get page title via page.title()
   - Get aria snapshot via page.locator('body').ariaSnapshot()
   - Get DOM summary via page.evaluate() with a recursive summarize function:
     - Max depth: 4 levels
     - Max total nodes: 500 (use a counter; set truncated=true when hit)
     - Skip script, style, svg, noscript tags
     - For each element: extract tag, id, classList, data-testid, role, aria-label, href (for links)
     - Text content: only for leaf nodes (no children), truncated to 100 chars
     - Selector generation: #id if has id, [data-testid="..."] if has data-testid, else tag.class1.class2
   - Close context in finally block
   - Return InspectResult { title, url, ariaSnapshot, domSummary, truncated }

2. Register shotput_inspect tool in src/server.ts:
   - Import inspectPage and InspectResult from ./inspect.js
   - Add server.tool("shotput_inspect", ...) after shotput_capture:
     - Description: "Inspect a web page's DOM structure to identify CSS selectors for element targeting. Returns structured DOM summary and accessibility tree. Use this before shotput_capture when you need to find the right CSS selector for an element."
     - Zod schema:
       - url: z.string().url()
       - width: z.number().int().positive().default(1280)
       - height: z.number().int().positive().default(720)
       - wait: z.union([z.enum(["networkidle", "domcontentloaded", "load"]), z.number().positive()]).default("networkidle")
       - timeout: z.number().positive().default(30000)
     - Handler: call inspectPage(params), return content array:
       - Text: "Page: {title} ({url})"
       - Text: "Aria Snapshot:\n{ariaSnapshot}"
       - Text: "DOM Summary:\n{JSON.stringify(domSummary, null, 2)}"
       - If truncated: Text: "Note: DOM summary was truncated. Large page - not all elements shown."
