<task type="auto">
  <name>Task 2: Implement element capture, page preparation pipeline, and update MCP schema</name>
  <files>src/capture.ts, src/server.ts</files>
  <action>
1. In src/capture.ts, add a preparePage function (called after navigation, before screenshot):
   ```
   async function preparePage(page: Page, params: CaptureParams): Promise<void>
   ```
   - Order: (1) hide elements via CSS `display: none !important`, (2) inject custom CSS via page.addStyleTag, (3) execute custom JS via page.evaluate
   - Each step only runs if the corresponding param is provided
   - Await each step sequentially

2. In captureScreenshot, after navigation and scroll, call preparePage(page, params).

3. Add element capture logic in captureScreenshot:
   - If params.selector is provided:
     - Use page.locator(selector) and await locator.waitFor({ state: 'visible', timeout: 5000 })
     - Validate: if omitBackground is true and format is "jpeg", set warning and force format to "png" (JPEG has no alpha channel)
     - If padding is 0 or undefined: use locator.screenshot({ type: format, quality, omitBackground })
     - If padding > 0: use locator.boundingBox(), compute clip rect with Math.max(0, ...) clamping, use page.screenshot({ clip, type: format, quality, omitBackground })
     - For CaptureResult width/height: return actual element dimensions (box.width + 2*padding, box.height + 2*padding) multiplied by scale, or for zero-padding use element's natural size
   - If params.selector is NOT provided: existing fullPage/viewport logic unchanged
   - If selector is provided, fullPage should be ignored
   - Wrap locator.waitFor in try/catch: if element not found, throw Error with message including the selector string

4. In src/server.ts, add new optional Zod fields to shotput_capture schema:
   - selector: z.string().optional().describe("CSS selector to capture a specific element")
   - padding: z.number().int().min(0).default(0).describe("Pixels of padding around element")
   - omitBackground: z.boolean().default(false).describe("Transparent background (PNG only)")
   - injectCSS: z.string().optional().describe("Custom CSS to inject before capture")
   - injectJS: z.string().optional().describe("Custom JavaScript to execute before capture")
   - hideSelectors: z.array(z.string()).optional().describe("CSS selectors of elements to hide before capture")
   - Map these new fields into captureParams object passed to captureScreenshot

5. Keep all existing tests passing. The new element capture tests should now pass (GREEN phase).
